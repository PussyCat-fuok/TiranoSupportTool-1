;プロローグ　星との決別

*夜の家
#
　楽しかったお祭りは終わり、家に帰りつく。
　空は今までより遠く見えた。星も遠く見えた。しかし、寂しいとは思わなかった。
　鍵を開けて、家に入る。いつも通り、月の光が自分の部屋に射しこんでいる。
　持っていた荷物はドアの近くに置き、勇喜はベッドに倒れた。
　ゆっくり仰向けになって、額に腕を載せる。
#勇喜
「楽しかった」
#
　噛み締めるような声は、ゆっくり消えていく。それでも、それだけだった。
　悲しくて、苦しくて、辛いだけだった昔は過去のことだった。
　今日、知った。
　やっぱり才能なんかない。
　それでも。それでも、もう寂しなかった。もう本当の夢を知った。
　きっと、夢を叶えるための道は決まっていた。
　噛み締めているものは、惜別と希望の矛盾した、友人とそれぞれの道へ別れる時に似た感情だった。
　きっとまた会える。だから、頑張れよ。
　そんな声が聞こえてくる。
　だから、もう心は決まっていた。

*電車の中
#
　透明な春の光が射す電車で、うつらうつらとサラリーマンが舟をこいでいる。隣では、ぴしっとスーツを着こなした女性が電車の振動に合わせて化粧をしていた。
　はたまた、スマートフォンをいじっている人、つり革をつかみ眠気に耐えている人、負けて立ったままに眠っている人、いろいろな人がいる。
　しかし、その誰もが楽しそうにしていない。仕事とはそういうものだろうか。
　仕事とは辛いだけなのだろうか。なら、どうして働くのか。お金のためだろうか。
　きっと誰もが楽しい仕事をしたいと願っている。それでも、誰もがそれぞれに楽しいと思うことを仕事にはできない。
　きっと僕は恵まれているんだろう。
　満員電車の中、押しつぶされながら車内を眺めていて思った。
　誰もができる仕事の何が楽しいのだろう。でも、僕だけしかできないものは何もない。小説も、音楽も、イラストも、どれもが誰も感動させることができなかった。
　なら、諦めるしかないだろう？
　決別することができれば、どんなに楽だっただろうか。しかし、創作の手は簡単には作者を離してはくれない。快楽を忘れさせてくれない。
　思えば、どうして僕なんかを社長は取ってくれたのだろう。弱気になっていたせいもあって、入ることに決めたけれど。
　小さく嘆息してしまう。前に立っていた中年男性にギロリとにらまれた。どうやら、嘆息も許してはくれないらしかった。小さく頭を下げると、男性はそっぽを向いた。
　はあ……。
　今度は心の中で息を吐く。

*夜の家
#
　帰宅した勇喜はスランプの脱出方法を検索していた。しかし、どれもイマイチ信憑性がない。
　そもそもスランプになった理由すら絞れていない。この状態では、スランプの脱出について調べてもしかたがないだろう。
　冷蔵庫からエナジードリンクを一缶、取り出す。プシュっと音を立てると、炭酸が湯気のように、白く立ち昇った。
　喉を炭酸が刺激しながら、体の奥へ沁みていく。
　どうして、花麦はーと先生はイラストを描くのだろう。
　強い気持ちがないと、創作は続かないことを勇喜は知っていた。外部からの影響で、創作が楽しくなくなることはよくあることだ。もしくは、金銭的なものもある。満足したから、というものもあるだろう。花麦はーとには当てはまらないだろうが。
　むしろ、花麦はーと先生は恵まれている。才能もファンもある。とうぜん、評価だって。
　いったい、どれだけのクリエイターが評価を得られずに消えていくのだろう。なにかを求めて創作をするのに、なにも得られないのなら、逆に中傷されるのなら、創作はしない。
　それでも、創作せずにはいられない。見てもらったという事実がクリエイターをむしばむ毒になる。
　勇喜は本棚を見た。小説の書き方、脚本の書き方、イラストの描き方、作曲の方法、他にも風景の写真集や辞典も収まっている。そのすべてに勇喜は目を通していた。しかし、完成したと胸を張れる作品はなかった。
　はあ、と溜息を吐く。
　夜空は雲が晴れて、星が瞬いている。勇喜は外へ出ることに決めた。

*夜の公園

　近くの公園へ着くと、勇喜はベンチに腰を下ろした。この公園のベンチから見る景色は、案外よく空を望めた。

　今度は地面近くから星空と公園が写るように撮る。電灯の朧な灯りと集まった蛾たちも一緒に写っていた。そこはかとなく、寂しさがある写真だった。

　文芸もイラストも音楽も辞めた今、最後に残ったものは写真だけ。でも、そこにあるものを切り取っているだけだからか、満たされない。
　ただ、自分でも綺麗な写真を撮れるかもしれない、とも思う。
　それは諦めきれない創作への感情だった。なにかを残したいという欲求だった。
　どうすれば、よいのだろう。
　悩みながら写真を撮った。今度は空だけを切り取った。点々と瞬く星がか弱く、月光の中に隠れてしまう。それでも、紺を煮詰めたような黒に負けまいと輝く白と、覆ううっすらとした靄が幻想的でもあった。そんなイラストになりえる加工前の写真だった。

　どうして創作をするのだろう。

#花麦

#勇喜

#花麦
「ああ、縹くんか」

　その質問に、花麦はーとは目を丸くする。

　遠くを見る花麦はーとの目に何が映っているのか、勇喜にはわからなかった。
#花麦
「それにね、上を目指す以上は苦しくなるのは当たり前だし、きっともっと上手になれば、もっと楽しく描けるようになると思うから、捨てたりはしないよ」
#
　花麦はーとが桜の木へ向かって遠くなっていく。その後ろ姿を勇喜は歯を食いしばって見ていた。

#花麦
「あ……」
#
　後ろからの声に、勇喜は顔を隠すように、遠ざかっていった。

*夜の家
#
　明かりもつけずに、勇喜はベッドに腰をかけていた。
　まだ胸はドキドキしている。たとえ意識していなかったとしても、たった一言で意識してしまう。
　勇喜の心に浮かぶのは、花麦はーとの笑顔だった。今まで見てきた作り笑いじゃない。仮面の下に潜めた本当の笑顔を見たい、そんなことを思っていた。
　今までは自分のことに気を取られていた。母の言葉で初めて他の人へ意識が向いた。
　ゆっくり今日のことを思い返せば、独りよがりの質問をしてしまった。
　明日、どうすればよいのだろう。
　今まで人の関わりが多くない勇喜にはわからなかった。
　謝って、それでどうすればいいのだろう。
　ただの励ましは意味がない。何か、きっかけを作らないといけない。
　どうすれば、また笑顔で絵を描いてもらえるのか。
　ふと、母の言葉が思い出される。
　恋人。
　恋人になりたい、というわけじゃないけれど、笑顔で描く姿を見たいな。
　笑顔になってもらうためにはどうすればよいのだろう。やっぱり、どこか、いつもとは違う場所に行ってみるのもよいかもしれない。
　なら、どこがよいだろう。
　父さんの小説を持っていたし、今までの作品を見ても、自然の多いところが好きなのかもしれない。
　自然の多いところ。それでいて、あまり遠くない場所。……公園だろうか。
　勇喜はスマートフォンで検索をかける。近くの、自然の多い公園に当てはまった場所は、散策にうってつけの場所だった。
　さっそく、スケジュールを組み始めた。

*夜の家
#
　家に帰ると、中は真っ暗だった。
　リビングに出ると、テーブルの上に書き置きがあることに気づく。

#母
『お疲れ様』

　その言葉を、勇喜は久しぶりに受け入れることができた。
　バッグを部屋に置き、シャワーを浴び、部屋着に着替える。冷蔵庫から麦茶を取り出し、息を吐いた。
　ゆったりと流れる時間を感じられるのはいつぶりだろう。今日は夕飯もいらないかな。

*居酒屋
#
　夏の長い日も、ついに夜の帳が下りる。勇喜と花麦はーと、社長、月兎は社長の奢りで打ち上げの居酒屋に来ていた。
#社長
「いやいや、今日は暑かったね。本当にお疲れ様。乾杯」
#勇喜・花麦・月兎
「「「乾杯」」」
#
　社長の音頭に、三人の掛け声がのる。大ジョッキの注がれた各々の酒が少しこぼれた。
　舟に乗って運ばれてくる唐揚げの山をＳＮＳに投稿しながら、社長は話を始める。
#社長
「今回のヒーローは佐藤くんかな」
#花麦
「私がスランプから抜けられたのも佐藤くんのおかげですし」
#勇喜
「え、いや、迷惑しかかけていないですし」
#
　社長と花麦はーとの称賛に、勇喜は全力で首を横に振った。
　なお、月兎は見かけに見合わず、黙々とハイペースで唐揚げを食べていく。その目は戦争をしていることを示していた。
　そんな月兎を放って、花麦はーとは勇喜を褒める。
#花麦
「それでも次のステージに手をかけることができたのは、間違いなく佐藤くんのおかげだよ」
#勇喜
「い、いや、そんな」
#
　それでも違うと首を振り続ける勇喜に社長はお礼を言う。
#社長
「佐藤くんのおかげで花麦は成長できたし、月兎くんも友だちを得ることができた。これからのクリエイター二人によい影響を与えることができたのは佐藤くんだからだよ。本当にありがとう」
#花麦
「私からも、ありがとう」
#
　社長と花麦はーとのお礼に、勇喜は助けを求めるように月兎を見た。
　しかし、そこには大ジョッキを空にして、赤くなった月兎が唐揚げの山の半分を崩している姿があった。
　これはダメだ。
　勇喜はすぐに顔を戻す。が、少し遅かった。
#月兎
「そういえば、勇喜くんは花麦先生とどこまで進んだの？」
#勇喜
「は」
#
　月兎の言葉に、勇喜は思わず声を漏らす。
#社長
「そんな仲にいつの間に？」
#
　社長が面白そうだとニヤニヤし始め、花麦はーとは頬を染める。
#花麦
「佐藤くんが泣いているのを抱きしめたり。胸に触ったりもしたよね」
#勇喜
「せ、先生！」
#
　満更でもなさそうな花麦はーとに、勇喜は悲鳴をあげた。
**
*夜の家
#
　社長に「嫁にもらってやってくれ」と言われたり、いやいやと断れば花麦はーとに悲しそうな表情をされるという地獄を味わった勇喜は、それでも酔いのおかげか明るい気持ちで帰る。
　家に帰りつくとベッドに倒れこむ。
　来ていたメッセージを開くと、社長からだった。
#社長
『佐藤くんはこれからどうしたい？　花麦の件から考えてもマネージャーという選択はありだと思うけれど。もちろん、佐藤くんがやりたい道を応援するよ』
#
　勇喜は『ありがとうございます。少し考えます』とだけ返信した。
　そうして、仰向けになって考える。
　やっぱり僕には才能がない。
　勇喜は両親の声が聞きたくなり、電話をかける。
#母
『お疲れ様』
#
　母の第一声はそれだった。
　その言葉だけで勇喜は泣きそうになった。
#勇喜
『ありがとう』
#
　少しの涙声のせいか、母の言葉はいつもより優しく感じた。
#母
『そうだ、勇喜の作品を買わせてよ。二人で読むから』
#勇喜
『え、そんな、データを今送るよ』
#母
『お金は後で払うよ。しっかり作ったのだから、受け取りなさい』
#勇喜
『わかったよ。ありがとう』
#母
『それで、どうしたの？』
#勇喜
『声が聞きたくなって』
#母
『そう。創作は続けるの？』
#
　直球な母の言葉に、勇喜は困ったように言った。
#勇喜
『もう辞めるよ』
#母
『そう。新しい夢を見つけたの？』
#勇喜
『言葉にまだできないんだ』
#母
『勇喜はどんなことがしたいの？　自分だけにしかできないこと？』
#勇喜
『うん』
#母
『そっか』
#
　考えているようで、母の言葉は止まる。そして、次の声は父の声だった。
#父
『勇喜、元気にしているか？』
#勇喜
『うん』
#父
『そうか』
#
　寡黙な父が声をかけるのは珍しかった。勇喜は脈拍が速くなる。
#父
『母さんから聞いた。創作は辞めるんだってな』
#勇喜
『うん』
#父
『頑張ったな』
#
　その言葉は勇喜にとって父にだけ、寡黙な父にだからこそ言われたかった言葉だった。
　いろはにぽぺとに入社してから、もう何度目ともしれない涙声で感謝する。
#勇喜
『ありがとう』
#父
『ああ』
#
　言葉は続かない。それでも、月を肴に酒盛りをしているような、心地よい静けさだった。
#父
『人は、自分だけにしかできないものを、宿命を、求め生きるものだ』
#勇喜
『うん』
#父
『創作は最たるものと思われているかもしれない。だから、憧れるのかもしれない。しかし、文を書こうが絵を描こうが音を奏でようが、けっきょく誰かの人生をよりよいものにできたものがよいのであれば、さして宿命に差があるわけじゃない』
#勇喜
『難しいよ』
#父
『そうか……。声で伝えるというのは難しいな。文の方がよっぽど簡単だ』
#
　苦笑しているのが、声でわかった。
#父
『人によりよい人生を送ってほしいと願って行動するのであれば、創作じゃなくてもできると言えばよいのか。確かに多くの人を自分が変えたいと思うのなら才能が必要かもしれないが、誰かを支えることで間接的に変えることができる、と言えばよいのか』
#
　勇喜の脳裏に、社長の言葉が過ぎる。花麦はーとや月兎のマネージャーができるのであれば勇喜の夢が叶うと知っての言葉だったのだろうか。
#勇喜
『ありがとう』
#父
『何かのきっかけになればよいが』
#勇喜
『なったよ』
#
　勇喜のはっきりした言葉に、父は優しく『そうか』と言って電話を切った。
#勇喜
「あ」
#
　母が怒る様子が目に浮かんだが、勇喜は知らないふりをすることにした。
　仰向けになって、空を眺める。太陽が昇っていた時と同じように、夜の空も晴れていた。
　本当に今日は楽しかった。
　才能がなくて苦しくても、今日は楽しかった。今日という日だけは楽しかった。
#勇喜
「楽しかった」
#
　もう一度、今度は口に出す。
　その声が体に染みこむ。
　でも、もうお別れだ。夢を託して支えよう。それが本当の夢だ。
　勇喜は目を閉じる。ゆっくり眠りに就いた。
　一筋の涙が夜の光を反射した。
**
*朝の道
#
　夏にしては珍しい、清々しい朝を迎える。
　勇喜は身支度を整えて、外に出た。風もあって、いつもよりかは過ごしやすそうだった。
　通知が来る。
#社長
『じゃあ、この間のところで』
#
　社長からだった。今日の外出は、社長への返事が目的だった。
　もう引目を感じることはなかった。あらゆるものが新鮮で色鮮やかだ。
　そんなことは最初からのはずなのに、気づけなかった。
　曇ったフィルターが外れて、ようやく素の世界が映る。
　勇喜は立ち止まる。日射しに照らされた建物の白と道路の色あせた赤に、木の緑は透明色のようにも映って、相まった光景は神秘的だった。
　勇喜はスマートフォンのカメラで一枚、撮った。
　そして、ＳＮＳに投稿する。
　この間の女性だろうか、すぐに一人が反応した。

*カフェ
#
　社長と待ち合わせた場所に来ると、社長はすでに待っていた。
#勇喜
「すみません。お待たせしてしまって」
#社長
「いや、今日はたまたま早くてね」
#
　そう笑う社長はいつも通りだった。
#社長
「それじゃ、カフェで話を聞こうか」
#勇喜
「はい」
#
　無言のままに、勇喜は社長についていく。
　わかりづらい場所にあるカフェに入ると、今日も客はいなかった。
#社長
「ご無沙汰だったね」
#
　社長が声をかけると、いつぞやのマスターは砕けた口調で挨拶をする。
#マスター
「ああ、久しぶりだ。その子はお前さんの会社の子だったんだな」
#社長
「世話になったよ」
#マスター
「何、いつも通りにもてなしただけさ。さてはあの嬢ちゃんもか」
#社長
「うちの売れっ子イラストレーター様さ」
#マスター
「大変だな」
#社長
「ああ、大変さ」
#
　置いてけぼりをくらった勇喜は、社長とマスターの顔を交互に見る。
#社長
「佐藤くん。この男は私の知り合いでね。まあ、いろいろ都合してもらったりしていたんだ」
#マスター
「こいつの依頼はいつも突拍子もないからな。そのせいか、けっこうこいつのコネは広いぞ。何かあったら、こいつに相談すればなんとかなるだろうさ」
#社長
「最初に言った通りさ。何かやりたいことがあれば、相談してくれよ。それで、今日は何かを決めたと言っていたけれど」
#勇喜
「あ、はい。マネージャーをしていきたいと決めました」
#社長
「そうか。じゃあ、これからしばらくは私について、いろいろと学んでいけばよいよ」
#マスター
「軽いなあ」
#
　マスターの言葉が勇喜の心情をよく表していた。重く考えていたせいで、あっけないように感じていた。
#社長
「だって、決意を尊重するって言っているんだから、何か言う必要はないでしょ」
#マスター
「まったく、人の想いは大切にするくせに合理的だからなあ」
#
　話はそれで終わりかい？　とでも言わんばかりに、社長はコーヒーを注文する。
#マスター
「お前さんはどうする？」
#勇喜
「あ、自分も同じもので」
#マスター
「承知した」
#
　苦笑しながら、マスターはキッチンへ向かう。
#社長
「それじゃ、今日はもう真面目な話ではなく夢の話をしよう」
#
　それからはもう互いの夢についてや、昨日のコミックマーケットの感想、花麦はーとや月兎に加え、他のクリエイターたちについてを話した。
　マスターを交えた会話は、とても楽しく、ようやく仲間になれたかのように感じる。
　そう勇喜は社長に言うと、社長は笑った。
#社長
「そんなのは今更だよ。私は君を仲間だと思っているし、一緒に仕事をしている戦友だと思っている。だから、これからも遠慮はしないでくれ」
#勇喜
「ありがとうございます」
#社長
「とうぜんだろ」
#
　そう言って、社長はコーヒーカップを持ち上げる。
#マスター
「おいおい、カップで乾杯をするつもりか？」
#社長
「ああ。人生はいつでも乾杯して飲み干すものさ。それにグラスだろうがジョッキだろうが、それこそコーヒーカップだろうが関係ないさ。紙コップで乾杯をしているんだから、それこそ今更ってやつさ」
#マスター
「なるほどな。確かにその通りだ」
#
　マスターもコーヒーカップを掲げて、あとは勇喜だけだった。
#社長
「さ、佐藤くんも」
#勇喜
「あ、はい！」
#
　勇喜も慌ててコーヒーカップを掲げる。
#社長
「それじゃ、夢に歩んでいくことに乾杯」
#マスター
「乾杯」
#勇喜
「乾杯です」
#
　カチャリと三つのコーヒーカップが鳴り、三人は一気にコーヒーを飲み干した。
　一気に飲み干すコーヒーの味は、いつもよりすっきりとしていた。

;エピローグ　花と星を支える

*いろはにぽぺと会議室
#
　お盆のお休みが終わり、最初の出勤日になる。
　勇喜が出社すると、すでに社長がオフィスにいた。
#勇喜
「おはようございます」
#社長
「おはよう。今日からよろしくね」
#勇喜
「はい」
#
　今日から勇喜は社長としばらく一緒に働くことになっていた。
　夏らしい、蒸し暑い空気と肌を焼く日射しが陽炎を見せる。
　まるで夏の短距離走でスタートを待つ心持ちだ。
　しかし、だからこそ夢に向かうスタートを切るのに、ふさわしい青空だった。
　勇喜は社長の後をついて、オフィスを出る。
　この一歩が勇喜にとっての、本当の夢を叶える最初の一歩だった。
